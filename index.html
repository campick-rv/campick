진행 방식
1

상담 문의

카라반 제조사·모델·연식, 희망 제품을 알려주세요. 전화 또는 카카오톡으로 문의 가능합니다.

2

차량 정보 확인

설치 가능 여부와 추가 작업 발생 여부를 사전에 확인합니다. 하부 사진을 미리 보내주시면 더 빠릅니다.

3

일정 조율 · 견적 확정

방문 날짜와 장소를 협의하고 최종 견적을 확정합니다. 전국 어디든 출장 가능합니다.

4

현장 방문 설치

약속된 날짜에 직접 방문해 설치합니다. 고객님이 자리를 비우셔도 괜찮습니다. 당일 완료 원칙입니다.

5

시운전 · 인도

실제 주차하시는 자리에서 작동 테스트 후 인도합니다.

제품 스펙은 평지 기준이며, 차량 하중·텅잭(전축하중)·노면 상태에 따라 결과가 달라질 수 있습니다.

Bluetooth Setup
블루투스 연결
처음 연결도 순서대로 따라만 하면 됩니다
⚠ 연결 전 반드시 확인

위치 권한 허용 · Bluetooth Classic 체크 해제 · 비밀번호 123456
이 세 가지가 가장 많이 막히는 포인트입니다.

STEP 1 — 앱 설치
1

앱 다운로드

안드로이드 Play스토어에서 rv리모컨 검색 후 RV 리모컨 (M+LAP) 설치

※ 애플 앱은 개발 중
STEP 2 — 앱 실행
1

앱 실행

RV리모컨 앱을 실행합니다.

2

위치 권한 허용

휴대폰 위치 권한을 반드시 허용해야 합니다. 허용하지 않으면 블루투스 기기 검색이 되지 않습니다.

앱 사용 중에만 허용 선택
STEP 3 — 기기 연결
1

Bluetooth Classic 체크 해제

화면 하단의 Bluetooth Classic 체크박스가 반드시 해제된 상태여야 합니다.

2

자동연결 선택

자동연결 옵션을 선택합니다.

3

블루투스 기기 선택

검색된 기기 목록에서 본인 차량 기기를 선택합니다.

예) RVMOVER-FSF253700
예) RVSAL-ESPF250100
4

비밀번호 입력

블루투스 연결 요청 팝업에서 비밀번호를 입력합니다.

비밀번호: 123456
5

연결 완료 확인

화면에 Bonding complete 메시지가 표시되면 연결 완료입니다.

6

새로고침

새로고침 버튼을 눌러 연결 상태를 확정합니다.

STEP 4 — 모델 설정
1

블루투스 등록 확인

연결된 기기 이름 옆에 (Default) 표시 확인 후 해당 기기를 선택합니다.

2

RVMover 모델 설정

사용하는 모델을 선택합니다. RV-SAL만 사용하는 경우 None 선택.

RV-9000AT / RV-9000MT / None
3

컨트롤 방식 선택

조작 방식을 선택합니다.

양손조이스틱 / 한손조이스틱 / 방향버튼
4

SAL 모델 설정

사용하는 SAL 모델을 선택합니다. RVMover만 사용하는 경우 None 선택.

SAL-Basic / SAL-Premium / None
5

설정 버튼 클릭

모든 설정 완료 후 설정 버튼을 눌러 저장합니다.

두 번째 연결부터는?
자동연결 옵션을 설정해뒀다면 앱 실행 시 자동으로 연결됩니다.
